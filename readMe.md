# check-pc-uptime

## Overview

check work hours and pc uptime.  
freee の勤怠日時と、LANSCOPE の PC 稼働時間を比較します。

## Requirement

- JavaScript

## Usage

１．チェックする日時及び差分許容時間の指定  
２．freee の勤怠日時読み込み  
３．LANSCOPE の PC 稼働時間ログ読み込み  
４．チェックを開始します
